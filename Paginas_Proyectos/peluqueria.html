Peluquería

Esta aplicación de peluquería está desarrollada en PHP sin utilizar frameworks como Laravel o Symfony. La estructura del proyecto sigue el patrón de diseño Modelo Vista Controlador (MVC), implementada de forma casera para mantener una mejor organización del código. La aplicación emplea JavaScript para algunas funcionalidades interactivas y cuenta con una API REST para la gestión de servicios. Utiliza Mailtrap para la gestión del registro de usuarios, validando el correo mediante tokens. Los estilos de la aplicación están automatizados utilizando Sass y Gulp, lo que permite una gestión eficiente y flexible del CSS. Los usuarios pueden registrarse, iniciar sesión y reservar servicios, seleccionando el tipo de servicio y el horario disponible. Los administradores pueden gestionar las citas, incluyendo la visualización y validación de las mismas. La aplicación proporciona una experiencia completa para la gestión de una peluquería, con funcionalidades adicionales que pueden ser implementadas en futuras actualizaciones.
