Blog

Este es un proyecto de blog creado en php puro y utilizando mysql como base de datos para almacenar cada artículo, para postear articulos de interés. Se pueden crear, editar y eliminar posts desde nuestro panel de admin, además de eso contamos con un panel de logueo.
