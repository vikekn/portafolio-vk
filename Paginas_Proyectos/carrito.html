Carrito de compras Angular

Esta aplicación fue creada en mis trabajos prácticos de mi carrera de Técnico en Desarrollo de Software durante el segundo año, la app tiene componentes bastantes obsoletos para las versiones actuales de Angular, pero fue una buena práctica para aprender a usar Angular y Firebase. La app tiene un Login y registro de usuario usando la API de fire base, el resto de componentes se crearon usando javascript para el carrito de compras y bootstrap para el diseño del carrusel. Para confirmar los pagos utilizaba la API de paypal, la cual era la unica de la que tenia conocimiento para implementar en ese momento.
