Sistema Ventas C# App Escritorio

Este es un proyecto de un Sistema para gestionar ventas, compras y el inventario de nuestro local. La aplicación permite añadir proveedores y clientes, gestionando toda la información relevante a través de una interfaz intuitiva y fácil de usar. Entre sus funcionalidades principales se encuentran la capacidad de generar facturas en formato PDF y generar reportes detallados en Excel. Está desarrollado en SQL Server y C#, y utiliza una arquitectura de capas. También es compatible con lectores de código de barras, lo que facilita la entrada de datos y la gestión eficiente del inventario, compras o ventas. Para la generación de PDFs utiliza iText7, y para la generación de reportes en Excel emplea ClosedXML. Esto facilita el seguimiento de las transacciones, el control de inventarios y la administración de relaciones con proveedores y clientes de manera eficiente. El sistema es una herramienta integral para cualquier negocio que busca optimizar sus procesos de ventas y compras.
