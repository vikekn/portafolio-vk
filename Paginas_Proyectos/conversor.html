Conversor

Este proyecto es un conversor de archivos multimedia desarrollado en Python. Utiliza ffmpeg para convertir archivos de video y extraer audio de ellos, brindando flexibilidad en la gestión de archivos multimedia. Para convertir imágenes, utiliza la librería PIL (Pillow), la cual maneja diversos formatos de imagen y ofrece opciones de conversión de alta calidad. La interfaz gráfica del usuario está construida con tkinter, lo que permite una interacción amigable y visualmente atractiva. El proyecto admite múltiples formatos de archivo: videos en mp4, avi, mov, mkv, flv y wmv; imágenes en jpg, jpeg, png, gif, bmp, tiff, webp e ico; y audios en mp3, wav, aac, flac, ogg y wma. Los usuarios pueden seleccionar un archivo, elegir el formato de salida deseado y realizar la conversión, todo desde una interfaz intuitiva. Además, se proporciona retroalimentación visual mediante cuadros de diálogo para informar sobre el éxito o cualquier error durante el proceso de conversión.
