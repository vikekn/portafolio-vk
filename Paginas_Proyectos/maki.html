Breve Explicación

Este es un proyecto de Asistente Virtual llamado Maki, el cual fue desarrollado en Python. Maki es un asistente virtual que puede realizar tareas como abrir aplicaciones, buscar información en la web, ver el clima y hora local, además de abrir videos en youtube. Maki es un proyecto en constante desarrollo, por lo que se le añaden nuevas funcionalidades en proximas actualizaciones.

Asistente Virtual Maki

El Asistente Virtual Maki es una herramienta desarrollada en Python que emplea diversas API y librerías para ofrecer sus funcionalidades. Utiliza la librería requests para realizar solicitudes HTTP y obtener información del clima a través de una API de clima. Usa pygame para reproducir el audio generado por Maki, permitiendo una fácil gestión del sonido. La librería speech_recognition proporciona soporte para el reconocimiento de voz, capturando y reconociendo los comandos hablados por el usuario. Para convertir texto en habla, Maki utiliza gTTS (Google Text-to-Speech), mejorando la experiencia del usuario mediante respuestas habladas. Además, se aprovecha de cohere para generar respuestas contextuales y conversacionales. Entre sus funcionalidades principales, Maki puede reconocer y responder comandos de voz, reproducir audio, buscar y abrir videos en YouTube, proporcionar información del clima y sostener conversaciones naturales usando IA. Ideal para una interacción fluida y amigable con los usuarios.
