Cajero Automático

Esta fue una aplicación educativa con la interfaz gráfica ya creada, para practicar y añadirle funcionalidades de un cajero automático. La aplicación permite autenticarse mediante un login en el que el cliente ingresa con su clave y DNI. Una vez autenticado, el usuario puede realizar varias funciones propias de un cajero: consultar saldo, consultar puntos, retirar saldo, canjear puntos, realizar transferencias y consultar el número de cuenta. La interfaz gráfica es intuitiva y fácil de usar, mostrando las opciones de transacción disponibles, así como una ranura etiquetada como "CASH" para la dispensación de dinero en efectivo. Este proyecto proporciona una excelente oportunidad para entender y desarrollar funcionalidades de un cajero automático en un entorno controlado y educativo.
